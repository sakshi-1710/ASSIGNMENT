My favourite subject is HTML. HTML is used to create the structure of web pages. It helps us add headings, paragraphs, images, and links to a website.


CSS is also one of my favourite subjects. CSS is used to design web pages. It helps us change colors, fonts, layouts, and make websites look beautiful.


JavaScript is very interesting because it makes websites interactive. With JavaScript, we can create buttons, forms, animations, and many dynamic features in a web page.
